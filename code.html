Webapp Code and Project Files

Code of the Binaural Beat Generator: I decided to share my code with an open-source license (the Unlicense, effectively public domain) because I believe binaural beats should be freely available for anyone to use as they see fit for their own benefit in terms of mental health, focus, sleep, or any other application.

The webapp itself (the binaural beat generator) is implemented as a single HTML file that contains all of the needed CSS and JavaScript to run in a standard web browser.

This overall project (the website) consists of:
  • WebApp page: index.html
  • Scientific background page: science.html
  • Code & downloads page: code.html
  • Optional masker audio files (e.g., Pink Noise.wav, Brown Noise.wav, etc.)
By downloading the project, you can run it on your local laptop/desktop by opening index.html in a web browser, ensuring offline access. On phones, running this code locally may work but has not been teseted.

Additionally, if you download the project, you can easily modify the code (manually or using AI) to substitute in any additional masker files that you prefer, assuming you prefer doing that to simply playing the pure tone and having a different window streaming music, video, or other sounds. Ultimately I anticipate that any long-term users of binaural beats for personal purposes will likely broaden their focus into binaural beat tracks made by professional creators with specific purposes, or simply layer the beats from an app like mine (or one capable of generating multiple beat frequencies at once) over any preferred masking sound on YouTube, a music app, or anywhere else.
